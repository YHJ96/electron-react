## Electron 설치

### Node

```
v18.17.0
```

### DEV 환경

```shell
npm install -g electron
// electron 버전 확인
electron -v
```

```shell
// React 실행 및 Electron 실행
npm run start
// npm run start 터미널 유지 한 채로 아래 명령어 dev 모드 진입
electron .
```

### Build

```shell
npm run build
```

dist 폴더 실행

- window -> exe
- mac -> arm64
